## 🚀 Installation

### Prerequisites
- Node.js 16+
- npm or yarn
- Git

### Clone Repository

```bash
git clone https://github.com/serinenadabenmissi-design/portfolio.git
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm start
```

Visit `http://localhost:3000`

### Build for Production

```bash
npm run build
```
